# クロコ導入 問診チェック

Claude Code（クロコ）デスクトップアプリ版 導入オフ会の受付用 Web アプリです。  
参加者がオフ会当日までに自分の環境を確認し、結果をチャットに貼り付けられる形式で提供します。

## 概要

- **対象**：クロコ（Claude Code デスクトップアプリ版）をインストールしたい方（完了済みも歓迎）
- **目的**：オフ会当日のインストール作業をスムーズに進めるための事前チェック

## 機能

- インストール状況を最初に確認（未着手 / 途中 / 完了）
  - 完了済みでも全設問を表示・下書き生成
  - 完了済みには活用方法・技術的なアドバイスには対応できない旨を案内
- Mac / Windows で質問が分岐
  - Mac：macOS バージョン確認（Tahoe / Sequoia以前）
  - Win：PC購入年確認（2022年以降 / 2021年以前）
- SMS受信・PCログイン・Googleアカウント・クレジットカードの確認
- 回答結果をチャット投稿用テキストとして自動生成・コピー

## デザイン

- ベースカラー：Claude Code オレンジ（`#DA7756`）
- スマートフォン対応（480px以下でボタン縦並び・タップ領域拡大）
- ボタンはアイコンなしのテキストのみ

## 技術スタック

- 純粋な HTML / CSS / JavaScript（フレームワークなし）
- 単一ファイル構成（`index.html`）
- 外部依存なし・ビルド不要

## デプロイ先

- **Cloudflare Pages**：`claudecode-offkai-checkin.miidacnt.workers.dev`
- GitHub への push で自動デプロイ

> コピーボタンは HTTPS 環境（デプロイ後）でのみ動作します。
