# クロコ導入 問診チェックアプリ

Claude Code（クロコ）導入オフ会の受付用 Web アプリです。  
参加者がオフ会当日までに自分の環境を確認し、結果をチャットに貼り付けられる形式で提供します。

## 概要

- **対象**：クロコ（Claude Code デスクトップアプリ版）をこれからインストールする初心者
- **目的**：オフ会当日のインストール作業をスムーズに進めるための事前チェック
- **主催**：IT活用コンサルタント サバ缶

## 機能

- Mac / Windows で質問が分岐
  - Mac：macOS バージョン確認（Tahoe / Sequoia以前）
  - Win：PC購入年確認（2022年以降 / 2021年以前）
- インストール状況確認（未着手 / 途中 / 完了）
  - 完了済みの場合は参加不可の旨を警告
- SMS受信・PCログイン・Googleアカウント・クレジットカードの確認
- 回答結果をチャット投稿用テキストとして自動生成・コピー

## 技術スタック

- 純粋な HTML / CSS / JavaScript（フレームワークなし）
- 単一ファイル構成（`index.html`）
- 外部依存なし・ビルド不要

## 使い方

`index.html` をブラウザで開くだけで動作します。  
GitHub Pages や Vercel などの静的ホスティングにデプロイして URL を参加者に共有してください。

> コピーボタンは HTTPS 環境（サーバーにデプロイ後）でのみ動作します。
