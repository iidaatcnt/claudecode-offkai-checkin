# 仕様書：クロコ導入 問診チェック

## 質問フロー

```
Q1. 端末確認（Mac / Windows）
  ├─ Mac → Q2a. macOS バージョン確認
  └─ Win → Q2b. PC購入年確認
              ↓
           Q3. インストール状況
             ├─ 完了 → ⛔ 警告（以降の質問なし）
             └─ 未着手 / 途中 → Q4〜Q7
                  Q4. SMS受信確認
                  Q5. PCログイン確認
                  Q6. Googleアカウント確認
                  Q7. クレジットカード確認
                       ↓
                  下書き自動生成 → コピーしてチャットへ
```

## 各質問の仕様

### Q1. 端末確認
| 選択肢 | 値 | ボタン色 | 備考 |
|---|---|---|---|
| Mac | `mac` | 緑 | リベシティ推奨のため緑 |
| Windows | `win` | オレンジ | — |

### Q2a. macOS バージョン（Mac選択時）
| 選択肢 | 値 | ボタン色 | アラート |
|---|---|---|---|
| Tahoe（macOS 15） | `tahoe` | 緑 | なし |
| Sequoia以前 / わからない | `old` | オレンジ | ⚠️ 警告（事前アップデートを促す） |

### Q2b. PC購入年（Win選択時）
| 選択肢 | 値 | ボタン色 | アラート |
|---|---|---|---|
| 2022年以降（4年以内） | `new` | 緑 | なし |
| 2021年以前（5年以上前） | `old` | オレンジ | ⚠️ 警告（スペック確認を促す） |

### Q3. インストール状況
| 選択肢 | 値 | ボタン色 | アラート |
|---|---|---|---|
| まだ何もしていない | `none` | 緑 | なし（このオフ会のメインターゲット） |
| 途中まで試した | `partial` | オレンジ | なし |
| インストール完了 | `done` | 紫 | ⛔ 参加不可の旨を通知（フロー終了） |

### Q4〜Q7（インストール未完了時のみ表示）
| 質問 | Yes | No |
|---|---|---|
| Q4. SMS受信 | 緑 | 赤（警告） |
| Q5. PCログイン | 緑 | 赤（警告） |
| Q6. Googleアカウント | 緑 | オレンジ（警告） |
| Q7. クレジットカード | 緑 | 赤（警告） |

## 下書き生成ルール

- 全質問に回答するとテキストエリアに結果文を自動生成
- 要確認項目があれば末尾に `⚠️ 要確認：〇〇` を追記
- インストール完了の場合は下書きを生成しない

## アラートカラー定義
| 種別 | 色 | 意味 |
|---|---|---|
| `ok` | 緑 | 問題なし |
| `warn` | オレンジ | 要確認・要対応 |
| `ng` | 赤 | NG・参加に支障あり |
| `stop` | 紫 | 参加不可 |

## デザイン仕様

- ベースカラー：`#DA7756`（Claude Code オレンジ）
- SP対応：480px以下でボタン縦並び、タップ領域拡大
- ボタンはアイコンなしのテキストのみ
